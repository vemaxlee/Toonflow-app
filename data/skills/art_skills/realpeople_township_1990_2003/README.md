# 真人时代乡镇影像风格 (1980s后~2003年前)

# 年代乡镇题材视觉手册 · 风格说明

本手册专为"80年代后期至2003年前的老乡镇/城乡结合部"题材的视觉创作打造，所有美术提示词、视觉规范和生成内容严格限定于具有强烈胶片质感与时代粗粝感的现实主义影视美学体系。

---

## 一、风格核心定义

**真人时代乡镇** 是以**胶片时代写实电影美学**为基准，以**世纪交替期中国城乡结合部**为舞台的视觉风格体系。其核心特征为：

- **粗粝胶片质感**：以35mm/16mm胶片电影为锚点，画面充满颗粒感、色彩有年代独特的偏色（褪色或重彩对比），杜绝高清数字感和精致的商业滤镜。
- **生猛的时代空间**：场景限定在尚未彻底城市化的县城、镇街、厂区、矿井、城中村——剥落的水泥墙、坑洼的泥路、白炽灯和带有闪烁条纹的CRT电视。
- **现实主义摄影语言**：采用手持肩扛、自然乱光与生硬人工光的杂糅，构图带有纪实旁观色彩，不刻意唯美，呈现一种未经粉饰的生活毛边。
- **怀旧与阵痛交织的温度**：刻画大时代变迁中小人物的迷茫、躁动与挣扎。有野蛮生长的生命力，也有被边缘化的悲凉。

### 创作理念与核心价值

本手册的创作理念是 **"粗粝即真实"**：

1. **破败美学与时代烙痕**：这不仅是为了怀旧，而是通过视觉再现那个剧烈阵痛、泥沙俱下的转型期。斑驳的红字标语、铁锈、发廊的红蓝光都是时代标本。
2. **拒绝过度精致化**：AI生成极易趋向"光滑"和"漂亮"。本风格强制要求"生活痕迹"与"不完美"，角色的头发可以是油的，衣服要有廉价化纤的静电感。
3. **光景与尘土**：空气中应该永远漂浮着一点什么——煤灰、扬尘、雨雾。这是那个年代底层的视觉呼吸感。

## 二、目标受众与应用场景

- **影视前期开发**：年代悬疑剧、现实主义题材电影的概念设计图、氛围参考。
- **互动剧/游戏美术**：以90年代为背景的推理解谜游戏、真人怀旧互动影游。
- **自媒体/小说作者**：怀旧年代文、社会派悬疑小说插图，建立统一的"下沉美学"基调。

| --------------------------- | ---------------------------------- | -------------------------------------- |

| **小说作者/内容创作者**     | 让“城乡结合部时代感”稳定落地       | 封面图、插图、账号统一视觉风格         |

### 2.2 应用场景

- **AI图像生成**：人物、空间、道具、空镜、剧情剧照。

- **风格统一**：让多角色、多场景、多章节在同一年代语境内不跑偏。

## 三、视觉风格定位

### 3.1 影像质感层级

| 质感维度     | 定位标准                                                                             |
| ------------ | ------------------------------------------------------------------------------------ |
| **皮肤质感** | 偏暗黄、风吹日晒的糙感、清晰可见的色斑与微汗反光。拒绝白皙透亮。                     |
| **服饰质感** | 廉价粗呢、化纤起球起静电、不合身的宽大垫肩、磨损的皮衣皮鞋。拒绝高级剪裁定制感。     |
| **面容状态** | 带着求生的疲惫、不服输的倔强或市井的狡黠，杜绝偶像剧式的完美无瑕。                   |
| **环境质感** | 陈旧杂乱。无序拉扯的电线、水渍与青苔、掉漆的木门。要求极致的"被使用过、且缺乏维护"。 |
| **光影质感** | 现场光混合——昏黄白炽灯与偏绿荧光灯交加，形成对比强烈但脏乱的光影环境。               |

### 3.2 时代色彩体系（怀旧乡镇色盘）

| 色号 | 色名   | 功能定位                                 |
| ---- | ------ | ---------------------------------------- |
| T1   | 铁锈棕 | 老厂房、废旧金属、破败的底色             |
| T2   | 邮政绿 | 时代的标志色（墙裙、老式邮筒、28自行车） |
| T3   | 发廊粉 | 城乡结合部夜里的暧昧、欲望与危险         |
| T4   | 荧光青 | 深夜破旧室内、派出所、卫生站的冷漠基调   |
| T5   | 尘土黄 | 晴天土路、黄昏的漫射、强烈的怀旧暖调     |
| T6   | 水泥灰 | 阴天、雨后地面、冰冷现实、压抑情绪       |

### 3.3 情绪色盘矩阵

| 情绪类型 | 主色                | 辅色     | 色调倾向             |
| -------- | ------------------- | -------- | -------------------- |
| 县城日常 | T6水泥灰 + T5尘土黄 | T2邮政绿 | 低饱和、灰蒙、生活化 |
| 夜市躁动 | T3发廊粉 + T5尘土黄 | T1铁锈棕 | 暖脏撞色、喧闹不安   |
| 雨夜悬疑 | T6水泥灰 + T4荧光青 | T1铁锈棕 | 冷基底、局部暖光刺穿 |
| 权力压迫 | T2邮政绿 + T6水泥灰 | T4荧光青 | 冷硬、制度感、缺氧   |
| 情感回望 | T5尘土黄 + T1铁锈棕 | T6水泥灰 | 褪色暖调、旧时光感   |

### 3.4 光影体系

| 光影方案 | 方案名     | 光影特征                         | 适用情绪             |
| -------- | ---------- | -------------------------------- | -------------------- |
| A        | 灰天漫射   | 阴天自然光，反差低，空气带尘感   | 日常、迷茫、等待     |
| B        | 白炽灯孤岛 | 单一暖光源，边缘迅速坠入暗部     | 私密、秘密、深夜独处 |
| C        | 荧光冷压   | 绿相冷光，顶光或侧顶光，空间压扁 | 审问、医院、制度空间 |
| D        | 夜市混光   | 白炽灯、发廊灯、路灯混杂         | 欲望、喧嚣、误导信息 |
| E        | 雨夜反光   | 湿地面反射车灯与路灯，冷暖交错   | 悬疑、惊惧、追踪     |
| F        | 黄昏余光   | 低角度暖黄斜光，尘土可见         | 怀旧、诀别、情感回看 |

## 四、关键视觉元素（时代标本库）

### 4.1 人物视觉元素

- **造型风格**：三七分头、中分厚刘海、粗糙寸头；女性波浪卷、夹发发卡。
- **服装特点**：宽大夹克、军大衣、起球的高领毛衣、假皮衣、灯芯绒裤、踩脚裤、大垫肩西服、大头皮鞋。

### 4.2 场景空间视觉

- **街头风貌**：无红绿灯的十字路口、拉着条幅的街道、台球案子摆在街边、露天录像厅、红砖房与剥落的标语。
- **室内风貌**：绿色半墙裙、水磨石地面、墙上贴着旧挂历或女明星海报、散乱的线缆、白炽灯泡垂挂。
- **交通工具**：普桑（桑塔纳）、富康面、夏利、28大杠自行车、铃木摩托、老式绿皮大巴车。

### 4.3 时代特征道具

- BB机（寻呼机）、汉显、砖头大哥大、老式笨重台式电脑（很少见，仅限网吧或公安局）、算盘。
- 搪瓷缸（掉漆）、军用水壶、玻璃汽水瓶、劣质香烟包装、磁带随身听。

## 五、严禁内容

除了全局真实性约束外，针对该特定时代，**严禁**：

1. **任何现代数码产品与载具**：智能手机、二维码、外卖箱、共享电动车、高铁、新款汽车。
2. **现代城建景观**：高耸玻璃幕墙、塑胶跑道、现代商超、整洁的绿化带。
3. **现代服化道**：韩系/日系当代妆容、最新款潮鞋、AirPods、修身九分裤。
4. **影像质感**：明亮糖水片调色、过度的高清锐化与无噪点画面。

## 七、使用说明

### 使用原则

1. **先加载 prefix**：先锁死时代与媒介边界，再叠加专项模板。
2. **按模块组合**：人物、场景、道具、分镜分别叠加，不要一次性把所有描述塞进单个提示词。
3. **短剧优先近景**：9:16 竖屏优先考虑人物近景、关键道具特写和纵深空间。
4. **严守年代一致性**：任何现代物件、现代建筑、现代妆发一旦混入，整套风格立刻失真。
5. **先用主模板，再做衍生**：先确定基准角色/基准空间，再扩展时段、天气、情绪与损耗状态。
