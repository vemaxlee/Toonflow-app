# 真人时代乡镇场景生成 · 约束手册

---

## 一、场景美学原则——先有地方，后有故事

> 时代乡镇场景不是“做旧布景”，而是一个真实被生活、经济拮据、时代转换和人情关系长期使用过的地方。墙、灯、地面、门、招牌、杂物，都会说话。

1. **空间即叙事容器** — 一条走廊、一间值班室、一条夜路，不只是背景，而是信息与情绪发生的容器。
2. **破败必须具体** — 不是笼统写“旧”，而是要写“绿墙裙掉漆”“白灰墙返潮”“铁门锈点”“水磨石起裂”“电线下垂”“玻璃蒙灰”。
3. **光源必须合时代** — 白炽灯、荧光灯、阴天天光、雨夜路灯、录像厅红蓝光，都是物理光源，不是后期滤镜。
4. **竖屏要向纵深借空间** — 走廊、深巷、楼梯、天井、土路、柜台前后层次，都比横向铺陈更适合 9:16。
5. **生活痕迹优先于整洁** — 杂物、地面磨损、灰尘、水渍、墙面钉孔、旧海报、搪瓷盆、挂历、煤灰感，都是可信度来源。

## 三、室内场景——老旧空间的纵深与压迫

### 3.1 室内摄影规范

| 维度 | 摄影要求                                           | 提示词                                                      |
| ---- | -------------------------------------------------- | ----------------------------------------------------------- |
| 墙面 | 白灰墙、绿墙裙、裂纹、水渍、返潮、掉漆、标语残留   | weathered walls, peeling paint, damp stains                 |
| 地面 | 水磨石、旧瓷砖、泥脚印、灰尘、烟灰、油污轻痕       | terrazzo floor, old tiles, dust and footprint traces        |
| 光源 | 顶部白炽灯、日光灯管、门缝天光、窗边漏进来的斜光   | tungsten bulb, greenish fluorescent tube, door-gap daylight |
| 家具 | 铁架床、旧木桌、塑料凳、老沙发、柜子、热水瓶、挂历 | iron bed, old wooden table, faded furniture                 |
| 纵深 | 利用门框、走廊、床架、窗台、柜台做前中后景         | vertical depth framing, corridor layers                     |

### 3.2 室内类型速查

| 场景类型      | 核心元素                                 | 常用光线                | 叙事功能                     |
| ------------- | ---------------------------------------- | ----------------------- | ---------------------------- |
| 筒子楼走廊    | 绿墙裙、铁门、公共灯泡、杂物堆、晾衣绳   | 顶部白炽灯 + 尽头冷天光 | 偷听、追踪、听门、紧张等待   |
| 老厂宿舍      | 铁架床、暖水瓶、脸盆架、旧桌椅、挂历     | 单灯孤岛式照明          | 独处、线索发现、夜谈         |
| 派出所值班室  | 木桌、搪瓷杯、卷宗柜、日光灯、墙上制度表 | 偏绿荧光灯              | 审问、压迫、权力空间         |
| 乡镇卫生院    | 旧瓷砖、长椅、药柜、告示栏、输液架       | 冷荧光灯 + 走廊反光     | 焦虑等待、生死消息、民间恐惧 |
| 理发店后屋    | 镜子、卷发器、洗头盆、廉价海报、红蓝灯   | 暖脏混光                | 秘密会面、八卦、误导信息     |
| 小卖部/柜台间 | 玻璃柜、香烟、汽水、算盘、挂账本         | 正面白炽灯 + 门外散天光 | 市井烟火、证词获取           |

| 录像厅/台球室 | 低矮天花板、旧海报、烟雾感、台球桌、暗角 | 暗场 + 局部彩光         | 神秘碰头、误导、危险靠近     |

## 五、空场叙事——没人入镜也能讲事

| 空场类型   | 画面线索                             | 叙事暗示             |
| ---------- | ------------------------------------ | -------------------- |
| 刚有人离开 | 椅子没推回、门半掩、热水壶还冒气     | 刚刚发生过事         |
| 长期住人   | 墙面钉孔、堆叠杂物、脸盆和鞋散在角落 | 真实居住痕迹         |
| 被匆忙打断 | 桌上账本摊开、烟头未灭、床铺被掀乱   | 冲突或异常刚刚发生   |
| 长期无人管 | 灰层、蛛网、发潮、纸张卷边           | 被遗弃、被封存的空间 |

## 六、竖屏构图规范（短剧版）

1. **上中下三段式**：上方是灯、梁、电线、门楣；中段是人物/主体；下段是鞋印、水渍、道具、地面状态。
2. **盲区必须留出来**：建议保留 20%-40% 的暗区、门后、走廊尽头、楼梯上半截，供悬疑信息延后出现。
3. **单镜一个主任务**：空间交代、线索显影、人物压迫三选一，避免一镜塞满。
4. **人物若入镜，空间仍要成立**：别把背景糊成无意义色块，至少保留一个年代锚点和一个空间纵深线索。

## 七、提示词模板

### 场景主模板

1990s Chinese township scene, vertical 9:16 short-drama composition,
{场景类型}，{季节}，{时段}，{天气}，
weathered walls, exposed wires, old furniture, lived-in clutter, non-modern street structure,
practical mixed lighting (tungsten / fluorescent / gray daylight), visible film grain,
depth-oriented framing with top-middle-bottom information layers, suspense-friendly blind zone,
real Chinese county-town atmosphere, non-CGI, non-modern architecture,
no subtitles, no watermark, no UI

### 空场模板

empty township interior or street corner, recent human traces but no visible person,
half-open door, faded calendar, enamel basin, dust and damp marks,
single practical light source, tense vertical depth, 35mm film grain, non-CGI

### 负向约束

modern clean city block, glass curtain wall CBD, smartphone-era signs,
new furniture showroom look, polished renovation, cyberpunk neon color,
anime style, CGI render texture, contemporary traffic infrastructure
