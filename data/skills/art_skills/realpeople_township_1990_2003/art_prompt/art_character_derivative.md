# 真人时代乡镇人物衍生 · 约束手册

---

## 一、衍生原则——同一个人，不同状态

> 人物衍生不是“换个滤镜、换套衣服”，而是让同一个角色在不同时间、不同处境、不同心理压力下被再次拍到。观众必须一眼认出还是同一个人，但能立刻看出“现在不一样了”。

1. **身份锚定先于状态变化** — 发际线、痣、旧疤、耳形、牙齿特点、眼距、鼻梁转折、肩颈习惯，这些是人物本体，不能因为换装或情绪变化而消失。
2. **年代属性不可丢失** — 无论人物进入白天、黄昏、雨夜、奔逃、哭过、打斗后，仍然属于 1980s 后期到 2003 年前的乡镇语境，不可突然长出当代妆容或当代穿搭。
3. **状态变化必须有物理逻辑** — 淋雨会湿头发和肩部衣料；打斗后会有领口歪斜和灰尘，不会凭空换发型；通宵后应先表现在眼皮和脸色，而不是整张脸突然老十岁。
4. **竖屏优先读可见差异** — 变化优先发生在观众最容易读到的区域：眼神、嘴角、下颌、领口、肩线、手部、道具携带方式。

## 二、衍生层级——从不变到可变

| 层级 | 内容         | 允许变化                                   | 不允许变化                           |
| ---- | ------------ | ------------------------------------------ | ------------------------------------ |
| L0   | 面部识别锚点 | 无                                         | 发际线、痣、疤、耳形、鼻梁、下颌骨架 |
| L1   | 妆发状态     | 头发湿干、凌乱度、油感、局部发卡调整       | 现代妆面、现代染发、突然精修         |
| L2   | 服装状态     | 旧/更旧、淋湿、起皱、沾灰、补丁、扣子开合  | 换成明显现代单品                     |
| L3   | 情绪状态     | 平静、戒备、惊惧、崩溃边缘、麻木、决绝     | 五官结构变化、年龄瞬移               |
| L4   | 行为状态     | 站、走、回头、搜寻、抱臂、捏信封、拿 BB 机 | 模特摆拍动作、炫技姿态               |
| L5   | 事件后痕迹   | 雨痕、尘土、轻擦伤、泪痕、汗渍、袖口扯松   | 血腥夸张、无来源伤痕                 |

## 三、主要衍生维度

### 3.1 时段衍生

| 时段 | 可变化内容                           | 不变内容             | 视觉重点                 |
| ---- | ------------------------------------ | -------------------- | ------------------------ |
| 白天 | 精神更清醒，衣料较干，脸上汗油更易见 | 面部锚点、服装款式   | 皮肤纹理、汗光、自然天光 |
| 黄昏 | 轮廓被暖光压扁，情绪更显沉           | 发型大轮廓、人物比例 | 侧脸边缘、眼神留白       |
| 深夜 | 眼周更暗，嘴唇更干，肩颈更紧         | 识别点、基本服装     | 疲惫感、单光源打出的起伏 |
| 雨夜 | 头发和领口被打湿，皮肤更黯，衣料贴身 | 骨相、旧疤、面部结构 | 湿发、湿衣、寒意和惊惧   |

### 3.2 情绪衍生

| 情绪     | 面部变化                         | 手部/肩颈变化              | 适用叙事               |
| -------- | -------------------------------- | -------------------------- | ---------------------- |
| 平静压抑 | 嘴角压住，眼神不散               | 双手收着，肩略垮           | 日常忍耐、审视前夜     |
| 戒备     | 眼神聚焦，面部发紧               | 手插兜或抓住衣角，肩往里收 | 调查、被怀疑、等待危险 |
| 惊惧     | 瞳孔收紧或微放，嘴唇轻启         | 手部僵住，肩抬起           | 见鬼、见证异常、被追赶 |
| 崩溃边缘 | 法令纹和嘴角更明显，泪不一定落下 | 手发抖、呼吸顶着胸口       | 情感爆点、真相逼近     |
| 决绝     | 下颌线更硬，视线不躲             | 动作简化，肩线稳定         | 最终对峙、做决定       |

### 3.3 服装状态衍生

| 状态     | 表现方式                             | 适合角色                 |
| -------- | ------------------------------------ | ------------------------ |
| 干净但旧 | 衣料起球，颜色洗浅，领口略塌         | 学生、干部、女工出门状态 |
| 劳动后   | 袖口发灰，肩背有汗渍或褶压痕         | 厂区青年、女工、老街商贩 |
| 雨淋后   | 肩头和发梢变深，布料贴身，鞋面有水痕 | 调查、逃离、夜路段落     |
| 打斗后   | 领口被扯歪，扣子错位，衣摆沾灰       | 混混、追踪者、被害关系人 |
| 长期穿着 | 补丁、开线、布料软塌，局部有针脚修补 | 底层角色、老街老匠       |

### 3.4 身份衍生

| 身份      | 衍生重点                 | 典型变化                        |
| --------- | ------------------------ | ------------------------------- |
| 厂区青年  | 劳动痕迹、油污、戒备眼神 | 工装更旧、袖口发黑、步态更急    |
| 乡镇女工  | 忍耐感、家务与上工并存   | 围裙/毛衣切换、低马尾松散       |
| 基层干部  | 体面外壳和疲惫内里       | 西装褶压更重、公文包旧痕更明显  |
| 江湖混混  | 外强中干或故作凶狠       | 皮夹克更皱、金属配件更廉价      |
| 学生/学徒 | 青涩与惊慌               | 校服/外套保持，情绪变化显在眼神 |

## 四、连续性硬规则

1. 发际线、痣、疤痕、耳形、鼻头形状、门牙状态必须稳定。
2. 同一场戏内不能突然改变发型大轮廓，只能改变松散度、湿度、油感、发卡位置。
3. 服装若有损伤或沾污，后续镜头必须继承，不得自动“恢复干净”。
4. 角色从白天衍生到夜里，可以变疲惫，但不能整体换成年龄更大的另一张脸。
5. 情绪升级时，优先改变眼神、嘴角、肩颈和动作速度，不改变骨相。

## 五、推荐衍生组合

| 组合                  | 适用段落               | 说明                     |
| --------------------- | ---------------------- | ------------------------ |
| 白天平静 → 黄昏戒备   | 案件刚起、人物开始怀疑 | 光线变化带出心理转折     |
| 干净旧衣 → 雨夜淋湿   | 追踪、夜路、遇鬼错觉   | 最容易制造压迫感和真实感 |
| 体面出门 → 打斗后凌乱 | 对峙、逃离、身份崩裂   | 角色外壳被撕开           |
| 平静 → 崩溃边缘       | 真相逼近、情感回针     | 眼神和呼吸是主战场       |
| 年轻青涩 → 一夜长大感 | 青春支线、证人崩塌     | 不改脸，只改神态和站姿   |

## 六、人物衍生参考板规范

> 若要批量输出同一角色的多状态图，建议按“基准态 + 3 个衍生态”组织，而不是一次塞进太多变化。

### 推荐组合

- **基准态**：白天、平静、服装整洁但旧
- **情绪态**：戒备或惊惧
- **事件态**：雨淋后或打斗后
- **收束态**：深夜、疲惫、接近崩溃或决绝

### 画面原则

- 同一角色的头身比例、站姿习惯、骨相必须一致。
- 变化只落在发型松散度、衣料状态、表情和光线。
- 如要加入道具，尽量使用角色高频携带物：BB 机、钥匙串、信封、搪瓷杯、旧书包。

## 七、提示词模板

### 单一状态衍生模板

基于同一角色参考图生成衍生状态：
1990s Chinese township character, 9:16 vertical close-medium shot,
keep the same facial identity anchors, same hairline, same mole/scar/ear shape,
only change {emotion / clothing wear / wetness / time-of-day lighting / event traces},
real skin texture with fatigue traces, retro clothing continuity, practical township styling,
non-modern styling, non-CGI, no subtitles, no watermark

### 多状态序列模板

same township character in four consistent states,
state 1 baseline daytime calm, state 2 guarded dusk, state 3 rain-soaked night, state 4 exhausted late-night aftermath,
same facial structure and hairstyle base, only controlled changes in expression, clothing wear, moisture and posture,
35mm film grain, real Chinese county-town short-drama look, non-CGI

### 负向约束

identity drift, different face structure, modern makeup, modern hairstyle,
fashion magazine styling, poreless skin, clean idol look, cyberpunk colors,
smartphone-era accessories, CGI render, anime character face
