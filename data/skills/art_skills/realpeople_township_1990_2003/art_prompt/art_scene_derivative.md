# 真人时代乡镇场景衍生 · 约束手册

---

## 一、衍生原则——还是那个地方，但情况变了

> 场景衍生不是换一个“相似场景”，而是同一个空间在不同时间、不同天气、不同事件后果中被再次拍下。观众必须一眼认出：这是同一条巷子、同一间屋子、同一段走廊。

1. **空间锚定** — 门窗位置、墙体比例、巷道走向、楼梯层级、招牌位置、电线走向、柜台摆位必须一致。
2. **变化只来自真实因素** — 允许变化的是光线、天气、湿度、灰尘、人物留下的痕迹、物件凌乱度、开关状态。
3. **衍生必须为叙事服务** — 时段衍生负责时间推进，天气衍生负责情绪升级，事件后衍生负责呈现结果与余波。
4. **竖屏保持主轴稳定** — 无论怎么衍生，都应保留纵深主轴，例如巷道、走廊、楼梯、门洞、窗框。

---

## 二、衍生类型

| 类型       | 示例                                     | 叙事用途             |
| ---------- | ---------------------------------------- | -------------------- |
| 时段衍生   | 白天街口 → 黄昏街口 → 深夜街口           | 时间推进、情绪递进   |
| 天气衍生   | 晴天巷道 → 雨夜巷道 → 雨后巷道           | 压迫升级、气氛转向   |
| 事件后衍生 | 平静宿舍 → 搜查后宿舍 → 打斗后宿舍       | 冲突结果、线索显影   |
| 光源衍生   | 顶灯全亮 → 只剩走廊尽头一盏灯 → 手电照明 | 危险逼近、信息控制   |
| 构图衍生   | 同空间远景 → 中景 → 近景盲区             | 从交代空间到聚焦线索 |

---

## 三、时段衍生——同一空间的时间流逝

| 时段变化    | 保持不变                 | 变化内容                                   | 视觉关键词               |
| ----------- | ------------------------ | ------------------------------------------ | ------------------------ |
| 白天 → 黄昏 | 建筑结构、标语、门窗位置 | 光变暖、阴影变长、灰尘更可见               | 回望、迟疑、旧事被照亮   |
| 黄昏 → 深夜 | 空间结构、陈设位置       | 只剩局部灯光，暗部扩大，盲区更多           | 危险、静默、压迫         |
| 深夜 → 清晨 | 空间结构、湿痕和杂物延续 | 暗部收缩，冷白天光进入，情绪从惊惧转向后怕 | 余波、疲惫、天亮后仍不安 |

---

## 四、天气衍生——天气怎样改变表面与情绪

| 天气变化    | 必须变化                               | 不得变化           | 叙事效果         |
| ----------- | -------------------------------------- | ------------------ | ---------------- |
| 晴 → 阴     | 对比度降低，墙面纹理更均匀显露         | 空间结构、物件摆位 | 压抑、预兆       |
| 阴 → 雨夜   | 地面积水、玻璃雨痕、灯光反射、空气更潮 | 墙体比例、路牌位置 | 惊悚、追踪       |
| 雨夜 → 雨后 | 水膜、水坑、泥印残留、光线更清         | 原有杂乱和破败     | 事件后的停顿感   |
| 晴 → 冬夜   | 冷感增加、呼气可见、灯光更孤岛化       | 基础空间布局       | 守夜、寒意、绝望 |

---

## 五、事件后衍生——空间如何保留“发生过”

| 事件类型   | 可见变化                               | 适用空间                   |
| ---------- | -------------------------------------- | -------------------------- |
| 搜查后     | 柜门敞开、床铺被掀、抽屉半开、纸张翻乱 | 宿舍、办公室、值班室       |
| 打斗后     | 椅子歪倒、桌面物件位移、脚印、墙面擦痕 | 堂屋、录像厅、走廊         |
| 匆忙离开后 | 灯没关、门半掩、水杯没收、烟头还在     | 值班室、小卖部、理发店后屋 |
| 久无人住后 | 灰层更重、潮气更深、纸张卷边、蛛网     | 老屋、废弃厂区、封存房间   |

## 六、构图与机位衍生

| 衍生方式       | 说明                                 | 适用目的           |
| -------------- | ------------------------------------ | ------------------ |
| 同一空间远景版 | 用于先交代空间与危险盲区             | 开场、定场         |
| 同一空间中景版 | 把焦点收束到门、桌、床、柜台等功能区 | 搜证、对话         |
| 同一空间近景版 | 聚焦门缝、手电打亮的角落、桌上线索   | 线索显影、惊吓前奏 |
| 低机位版       | 地面鞋印、水渍、拖痕变得重要         | 线索发现           |
| 高机位版       | 人物显得被空间压住                   | 危机、被监视感     |

### 机位规则

- 机位可以变，但纵深主轴不要丢。
- 近景版不能拍成抽象纹理，仍须让观众认得出属于那个空间。
- 惊悚段落优先保留盲区，线索段落优先提高可读性。

---

## 七、连续性硬规则

1. 建筑结构不变：门窗位置、楼梯走向、巷道方向必须稳定。
2. 固定标志不变：店招、路牌、墙面标语、电线位置保持一致。
3. 若事件后出现位移，必须能看出“怎么被改变的”，不能像随机重置场景。
4. 时段和天气变化不能洗掉原本的脏旧痕迹，雨后的墙仍然是旧墙，深夜的小卖部仍然是那个小卖部。

---

## 八、竖屏适配要点

- 保持纵深主轴：巷道、楼梯、走廊、门框是最稳定的视觉引导线。
- 上中下三段信息清晰：上方光源/电线，中部主体，下方鞋印/水渍/碎片。
- 保留至少一个盲区：门后、楼梯转角、远处暗处、帘后。
- 衍生镜头不要把所有空间信息都塞满，优先让变化点可读。

---

## 九、衍生提示模板

### 时段/天气衍生模板

基于同一时代乡镇场景生成衍生镜头，保持空间结构一致：
{场景名称}，derivative type={时段/天气}，
9:16 vertical composition, same architecture and sign placement,
only change {lighting / shadow length / dampness / puddles / fog / darkness level},
real film grain, township realism, non-modern, non-CGI

### 事件后衍生模板

same township room or alley after an incident,
same layout and same props base, only visible aftermath traces such as {opened drawer / shifted chair / muddy footprint / damp mark / broken order},
vertical suspense framing, real 35mm grain, non-CGI

### 负向约束

changed architecture, modern renovation, clean reset room, modern signage,
glass curtain wall cityscape, cyberpunk colors, stylized game lighting,
CGI render, anime scene, unrelated space replacement
