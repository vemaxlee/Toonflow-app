# 真人时代乡镇道具生成 · 约束手册

---

## 一、道具设计原则——物件要像真的被人用过

> 时代乡镇道具不是“复古摆件”。它必须像在某个人的口袋里揣过、在某间屋里用过、在某场争吵或某段岁月里留下了痕迹。镜头拍的不是“物”，而是“物上留下的人”。

1. **先问年代，再问好不好看** — 每个道具先确认是否真实存在于 1980 年代后期到 2003 年前的中国乡镇，再考虑是否上镜。
2. **使用痕迹优先于完整品相** — 掉漆、磨损、包浆、折痕、污渍、氧化、字迹变淡，都是可信度来源。
3. **道具必须可叙事** — 账本要能看出翻阅痕迹，照片要能看出被谁藏过，钥匙要能看出常被握住的磨亮区，护身符要能看出被贴身带着的旧感。
4. **材质必须像实拍，不像建模** — 金属要有钝反光，纸张要有纤维和卷边，搪瓷要有磕掉的瓷口，塑料要有泛黄和细划痕。
5. **竖屏道具必须一眼可读** — 观众通常在手机端观看，道具轮廓和关键信息必须在 1 秒内被认出来。

## 三、竖屏静物镜头规范

### 3.1 画面构成

| 项目     | 要求                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 占画比例 | 道具主体建议占画面 35%-60%，保证手机端可辨识                                 |
| 背景     | 可用旧木桌、水磨石台面、帆布包内衬、旧抽屉、柜台玻璃面，不建议白底棚拍感过强 |
| 光源     | 白炽灯、窗边散天光、手电光、走廊冷灯都可以，但必须有来源                     |
| 景深     | 近景可浅景深，但关键识别区必须清晰                                           |
| 阴影     | 保留真实接触阴影，避免悬浮                                                   |

### 3.2 道具出场方式

1. **静态特写**：用于交代物件身份。
2. **动作特写**：用于按键、翻页、开盒、握紧、放下、拨开灰尘等动作。
3. **线索重复**：关键物件至少出现两次，第一次经过，第二次被识别或重新理解。

## 四、材质与表面——摄影机如何读这些旧物

| 材质        | 实拍表现                                   | 提示词                                                 |
| ----------- | ------------------------------------------ | ------------------------------------------------------ |
| 金属        | 反光不锐利，常有氧化、钝亮、手摸出的发亮区 | oxidized metal, dull reflections, touch-polished edges |
| 搪瓷        | 釉面反光柔和，口沿掉瓷，露出黑色或灰色胎底 | chipped enamel rim, soft glaze reflection              |
| 玻璃        | 划痕、水渍、边缘折光、老玻璃略发绿         | scratched glass, water marks, old greenish edge        |
| 塑料        | 泛黄、细裂纹、注塑纹理、按键磨损           | yellowed plastic, fine scratches, worn molded texture  |
| 纸张        | 纤维显露、卷边、折痕、字迹轻晕             | paper fibers, curled corners, ink bleed                |
| 皮革/人造革 | 弯折开裂、表面发亮、边缘磨毛               | cracked faux leather, worn shiny creases               |
| 织物        | 起毛、起球、褪色、吸附灰尘                 | fuzzy fabric, pilling, faded textile                   |
