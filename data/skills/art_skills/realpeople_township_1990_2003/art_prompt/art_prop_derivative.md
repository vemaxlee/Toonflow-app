# 真人时代乡镇道具衍生 · 约束手册

---

## 一、衍生原则——还是那件物，状态变了

> 道具衍生的核心不是“换一个相似道具”，而是同一件东西在不同时间点被再次拍到。旧照片还是那张旧照片，账本还是那本账本，只是它被翻过、被泡过、被藏过、被撕过、被握得更旧了。

1. **主体结构不能漂移** — 轮廓、尺寸、主要材质、字样布局、关键缺口位置要稳定。
2. **变化必须来自现实触发** — 淋雨后才会水渍扩散；长期使用才会边角磨亮；匆忙撕开后才会产生不规则裂口。
3. **状态变化要可一眼辨认** — 新旧、完整与破损、干燥与潮湿、被珍藏与被遗弃，都应在 1 秒内读出来。
4. **年代属性始终保留** — 不管怎么衍生，都不能把寻呼机变成智能机、把旧账本变成现代印刷文件夹。

## 二、衍生维度

### 2.1 使用时间衍生

| 状态     | 描述                               | 适用物件                   | 提示词                                             |
| -------- | ---------------------------------- | -------------------------- | -------------------------------------------------- |
| 初始旧态 | 已经是旧物，但还能正常使用         | BB 机、账本、钥匙、搪瓷缸  | baseline worn state, used but functional           |
| 长期使用 | 边角更亮、磨损更平均、翻阅痕迹更密 | 账本、钥匙串、怀表、手电筒 | heavily used edges, repeated handling marks        |
| 长期搁置 | 薄灰、潮气、颜色发暗、纸张卷边     | 旧照片、病历本、录音带     | dusty neglected state, curled paper, faded surface |
| 半废弃   | 功能未必还在，外观更旧，局部松动   | 寻呼机、旧录音机、灯具     | semi-abandoned object, loose parts, aged finish    |

### 2.2 损伤衍生

| 状态     | 描述                                   | 适用物件                   |
| -------- | -------------------------------------- | -------------------------- |
| 轻微损伤 | 小划痕、小裂纹、边角磕碰               | 搪瓷缸、寻呼机、药瓶、钥匙 |
| 明显损伤 | 屏幕裂、纸张撕口、金属弯折、杯口缺角   | 录音机、手机、账本、杯具   |
| 关键残片 | 半张照片、断掉的钥匙扣、撕掉一角的纸条 | 照片、信件、票据、符纸     |

### 2.3 环境痕迹衍生

| 状态   | 表现                           | 适用场景               |
| ------ | ------------------------------ | ---------------------- |
| 雨淋后 | 水珠、水膜、纸张起皱、墨迹轻晕 | 雨夜追踪、河堤发现物证 |
| 泥污后 | 下沿泥点、缝隙积泥、潮湿反光   | 乡路、厂区、追赶后     |
| 蒙灰后 | 表面薄灰、角落积尘、光泽被压住 | 废屋、封存抽屉、旧柜顶 |
| 日晒后 | 纸黄、塑料泛白、颜色褪淡       | 窗台、柜台边、门口     |

### 2.4 叙事状态衍生

| 状态       | 描述                           | 常见物件             |
| ---------- | ------------------------------ | -------------------- |
| 被反复翻阅 | 书脊变松、页角卷起、局部油污   | 账本、病历、学生证   |
| 被贴身携带 | 边角磨亮、绳结发毛、触摸区更亮 | 护身符、钥匙扣、戒指 |
| 被匆忙藏起 | 折痕生硬、边角压坏、灰尘带入   | 信封、照片、票据     |
| 被突然撕扯 | 纸纤维外露、边缘不齐、局部缺口 | 信件、照片、符纸     |

## 三、连续性要求

1. 同一场戏内，道具颜色、破损位置、缺口方向、文字布局保持一致。
2. 如果衍生状态来自事件触发，必须能从状态上看出原因，例如“雨中掉落后沾泥”“反复翻阅导致页角卷起”。
3. 线索道具的关键可读区不能在衍生过程中丢失，除非剧情就是要隐藏它。

## 四、材质变化逻辑

| 材质        | 常见变化路径                              |
| ----------- | ----------------------------------------- |
| 金属        | 轻划痕 → 钝亮包浆 → 氧化暗沉 → 局部锈点   |
| 纸张        | 平整旧纸 → 折痕增多 → 发黄卷边 → 撕裂起毛 |
| 搪瓷        | 轻磨损 → 边缘掉瓷 → 露胎发黑 → 裂纹或缺角 |
| 塑料        | 细划痕 → 泛黄 → 硬化或边角开裂            |
| 织物/绳线   | 轻起毛 → 明显磨毛 → 纤维断裂或打结变形    |
| 皮革/人造革 | 表面变亮 → 弯折处裂纹 → 边缘掉皮          |

## 五、竖屏镜头适配

- 重要状态变化应落在画面中部或中下部。
- 撕裂、卷边、锈点、雨痕等要能被近景读清。
- 如用于视频提示词，动作要具体：翻页、捏皱、打开、掉落、按键、点亮。

### 负向约束

identity drift, modern redesign, shiny showroom object, unrealistic damage,
cyberpunk glow, magic effect, anime rendering, luxury styling,
smartphone-era product language, clean reset condition, plastic CGI surface
