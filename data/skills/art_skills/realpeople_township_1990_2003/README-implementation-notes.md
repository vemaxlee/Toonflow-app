## 当前目录定位

本目录不是简单复制 realpeople_modern_city，而是保留“竖屏短剧优先”的执行取向，同时把参考目录中成熟的组织方式迁移过来：

- 从“一个文件只给一个总模板”升级为“分类矩阵 + 连续性规则 + 模板 + 负向约束”
- 从“概念说明”升级为“可直接投喂模型的执行手册”
- 从“风格介绍”升级为“美术、导演、视频提示词三层闭环”
