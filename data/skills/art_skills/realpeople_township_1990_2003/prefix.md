## 二、摄影机语法（替代"渲染引擎"）

> 本类型不依赖渲染管线。画面品质取决于反映时代特质的摄影机选择、镜头的性格、光线的调度。

### 2.1 画幅与格式

| 参数     | 基准值                               | 美学意图                                                                                 |
| -------- | ------------------------------------ | ---------------------------------------------------------------------------------------- |
| 画幅比   | 9:16（竖屏短剧特化）                 | 构图重心由于横变竖，侧重人物中近景及微表情，环境信息做纵向层次铺展；保留粗糙的年代感边缘 |
| 影像格式 | 35mm胶片 / 16mm胶片质感              | 明显的颗粒感、偏暖或偏绿的轻微色偏、高光略微溢出的光晕                                   |
| 色彩科学 | Kodak Vision3 或 Fuji 电影卷偏色复刻 | 肤色偏黄暖、暗部偏绿或偏青、具有时代的褪色感                                             |

### 2.2 镜头性格

| 焦段                   | 叙事性格                           | 典型应用                                   |
| ---------------------- | ---------------------------------- | ------------------------------------------ |
| 28-35mm（广角/小广角） | 纪实感、环境吞没人物、粗粝的现场感 | 破败的街道跟拍、逼仄的筒子楼室内、集市人群 |
| 50mm（标准）           | 客观凝视、不加粉饰的日常           | 人物肖像、在小饭馆的对坐、路边的等待       |
| 85mm（中长焦）         | 情绪聚焦、背景虚化将人从杂乱中剥离 | 情感宣泄时刻、发现真相的惊愕、特写         |
| 135mm+（长焦）         | 窥视感、远距离的压迫与疏离         | 巷子尽头的暗中观察、矿区远眺               |

### 2.3 机位与运动

| 手法                 | 视觉心理                             | 使用场景                               |
| -------------------- | ------------------------------------ | -------------------------------------- |
| 固定长镜头           | 冷静、滞重感、时代的旁观者           | 破败工厂的全景、老屋里的静坐、街角发呆 |
| 肩扛晃动             | 现场感、呼吸感、底层的生猛与不安     | 追逐打斗、迷茫地游走、争吵对峙         |

## 三、时代乡镇色谱——光与材料的年代记忆

> 色彩来自世纪之交中国城乡结合部真实的光与生猛的材料，绝无精致的滤镜感。

### 3.1 时代光源色谱

| 光源            | 色温（K） | 光色描述                             | 乡镇场域                                  |
| --------------- | --------- | ------------------------------------ | ----------------------------------------- |
| 晨光与黄昏      | 3000-4000 | 偏黄的漫射光，空气中有煤烟和扬尘质感 | 早市、下班的厂区路、农田与土路            |
| 老式白炽灯      | 2500-2800 | 刺眼的昏黄光晕，照射范围有限         | 筒子楼楼道、破旧出租屋、路边台球室        |
| 荧光灯管        | 4000-6000 | 偏绿的惨白冷光，常伴有频闪           | 镇上卫生院、深夜小卖部、派出所预审室      |
| CRT电视屏幕光   | 变化      | 幽幽的蓝紫或冷绿光，带着扫描线的闪烁 | 昏暗的客厅、录像厅、深夜看更房            |
| 斑驳霓虹/发廊灯 | ——        | 刺眼但不纯正的粉红、大红或廉价蓝     | 镇上的理发店、歌舞厅、夜市大排档          |
| 卤素车灯        | 3000-3500 | 穿透雨雾的暖黄光束，光效重           | 泥泞土路上的桑塔纳/吉普车、破旧摩托车大灯 |

### 3.2 时代材料色谱

| 材料            | 色域                                         | 在乡镇中的位置                               |

## 四、全局约束规则

### 4.1 影像真实性与时代感约束（必守）

| 编号 | 规则                                                                                                                      |
| ---- | ------------------------------------------------------------------------------------------------------------------------- |
| P1   | 必须以「真人实拍 / 胶片电影剧照」为生成锚——画面必须有老照片或老电影的时代做旧感。                                         |
| P2   | 必须体现年代特征器物：BB机(寻呼机)、大哥大、老式诺基亚、二八大杠自行车、桑塔纳、小面、搪瓷缸。                            |
| P3   | 人物造型必须符合年代语境：宽大的西服套装、垫肩、皮衣、化纤起球毛衣、牛仔外套、健美裤。拒绝现代修身剪裁和精致布料。        |
| P4   | 场景必须具有"破败感与未规划感"：杂乱乱拉的电线、随意张贴的小广告和红字标语、坑洼不平的柏油/黄土路面。杜绝现代化整洁街道。 |
| P5   | 人物面容与状态必须真实：皮肤泛黄/粗糙、头发可能油腻或带有不打理的杂乱、微表情带有生存的疲惫感。杜绝白幼瘦与精致妆容。     |
| P6   | 字体与标牌：如有出现文字元素，必须是手写红底白字板报体、掉漆的宋体/楷体店招。                                             |
| P7   | 色彩呈现应有褪色或不饱和倾向：避免高饱和的现代数字影像色彩表现。                                                          |

### 4.2 摄影品质约束（必守）

| 编号 | 规则                                                                                   |
| ---- | -------------------------------------------------------------------------------------- |
| Q1   | 必须声明摄影机的在场方式：固定长镜头 / 肩扛晃动 / 纵向缓推 / 长焦窥视。                |
| Q2   | 竖屏景深必须有叙事意图：近景突出人物微表情，道具特写突出线索，深景深用于交代逼仄空间。 |
| Q3   | 高光可以有轻微溢出，但不能死白；暗部可以深，但必须保留结构和轮廓。                     |
| Q4   | 必须声明颗粒偏好：16mm粗颗粒 / 35mm细颗粒，严禁“高清无噪点数字广告感”。                |
| Q5   | 冷暖混光必须来自真实光源，不允许无来源的彩色氛围光和滤镜式霓虹染色。                   |
| Q6   | 竖屏构图优先中近景和纵向层次，避免横向信息过载；主体必须在手机端一眼可读。             |
