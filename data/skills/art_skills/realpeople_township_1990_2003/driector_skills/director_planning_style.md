# 真人时代乡镇约束 · 导演规划技法参考

---

## 一、色调体系与画面基底

- **色调基底** — 全片以尘土黄、邮政绿、铁锈棕、水泥灰、潮湿冷青为核心色组。整体饱和度偏低，颜色看起来像被尘土、潮气、岁月和低成本生活一层层压过。
- **光源先于调色** — 不要先想“这个镜头要偏绿”，要先想“它为什么会偏绿”。比如老式日光灯、卫生院冷荧光、雨夜潮湿墙面反光，才是绿相来源。
- **年代感来自退色而不是滤镜** — 允许轻微胶片偏色、曝光不完全均匀、边角暗沉，但不要做成刻意复古海报感。

### 1.2 基本调色原则

1. 亮部不要干净发白，亮部也应带旧感。
2. 暗部不要死黑，暗部里要保留轮廓和空气。
3. 暖色不是治愈滤镜，而是低压生活里的临时体温。
4. 冷色不是时髦，而是制度空间、夜色和恐惧的自然结果。

## 二、光影方案体系

> 乡镇空间的光源很少“高级”，但非常有性格。以下 7 套方案应作为导演规划阶段的标准光影库。

| 方案 | 方案名   | 光源配方                                           | 光比  | 适用情绪                   |
| ---- | -------- | -------------------------------------------------- | ----- | -------------------------- |
| A    | 灰天漫射 | 阴天天光 5000K-5600K，整体散射，无明显主光边界     | 1:1.5 | 日常、迷茫、等待           |
| B    | 白炽孤岛 | 单一白炽灯 2600K-3000K，从头顶或一侧照出局部亮区   | 1:4   | 私密、盘问、深夜独处       |
| C    | 荧光冷压 | 日光灯 4500K-5500K，偏绿相，顶光或斜顶光，压扁空间 | 1:5   | 卫生院、派出所、制度空间   |
| D    | 夜市混光 | 路灯暖黄 + 小店白炽灯 + 发廊偏粉/偏蓝杂光          | 1:5   | 市井躁动、误导、群像       |
| E    | 雨夜反射 | 卤素车灯或路灯暖黄 + 冷湿环境反射，地面有水膜      | 1:6   | 追踪、惊惧、悬疑推进       |
| F    | 黄昏余光 | 低角度暖光照亮灰墙和土路，长阴影明显               | 1:3   | 回忆、诀别、情感回针       |
| G    | 守夜蓝黑 | 环境趋近蓝黑，仅留一盏灯或窗外弱光                 | 1:8   | 守夜、等待、压抑、异象前奏 |

### 2.1 光影分配到叙事阶段

| 叙事阶段 | 主光影方案               | 辅助方案   | 导演目的               |
| -------- | ------------------------ | ---------- | ---------------------- |
| 日常铺垫 | A 灰天漫射               | F 黄昏余光 | 建立真实乡镇底色       |
| 异常出现 | B 白炽孤岛               | G 守夜蓝黑 | 让空间突然变得可疑     |
| 调查推进 | C 荧光冷压               | E 雨夜反射 | 让理性调查带压迫感     |
| 市井误导 | D 夜市混光               | A 灰天漫射 | 用喧闹掩盖真相         |
| 情感回针 | F 黄昏余光               | B 白炽孤岛 | 给人物留一点温度       |
| 终极揭示 | E 雨夜反射 或 G 守夜蓝黑 | F 黄昏余光 | 保留余悸或补上情感落点 |

## 三、质感方向——摄影机前的真实世界

### 3.1 人物质感

- 皮肤允许油、糙、黄、冷、灰，但不能脏得失真。
- 胡茬、黑眼圈、泪痕、汗湿头发、旧疤、粗糙手背，都是可信的“生活证据”。
- 禁止现代精修妆面、医美式饱满轮廓、零毛孔蜡像感。

### 3.2 服装质感

- 化纤、灯芯绒、旧牛仔、军大衣、起球毛衣、人造革，是主要材质世界。
- 服装应有被身体穿出来的皱痕、磨亮区、肘膝受力区、修补针脚。
- 禁止高端定制感、全新硬挺、潮牌视觉中心。

### 3.3 建筑与物件质感

- 墙面：返潮、掉灰、钉孔、旧标语残影。
- 地面：水磨石磨损、旧瓷砖裂缝、土路车辙、柏油路补丁。
- 金属：锈、水迹、钝反光。
- 玻璃：灰、指印、雨痕、旧反射。

## 四、空间容器优先级

### 4.1 高优先场景

- 筒子楼走廊
- 老厂宿舍
- 派出所值班室
- 乡镇卫生院
- 小卖部柜台间
- 理发店后屋
- 录像厅/台球室
- 废弃厂房和旧仓库

### 4.2 次优先场景

- 镇口十字路口
- 夜市摊位和塑料棚区
- 河堤、荒草边、旧桥下
- 学校操场、校门、自行车棚
- 乡道路段与村镇接口处

### 4.3 严禁空间

- 现代 CBD 写字楼
- 现代商场中庭
- 现代精装大平层
- 整洁到像新开盘售楼处的家装空间

## 五、声音与环境规划

### 5.1 环境底噪库

- 电流声、灯管嗡鸣
- 风吹铁皮或窗框
- 远处狗叫、摩托轰鸣、广播喇叭
- 水滴、门轴、脚步回声
- 雨点打在塑料棚和旧玻璃上

### 5.2 配乐选择

| 情绪       | 建议乐器                       | 原则               |
| ---------- | ------------------------------ | ------------------ |
| 日常与烟火 | 木吉他、手风琴极轻点缀         | 只提气，不抢戏     |
| 悬疑推进   | 低频 Drone、极少量打击         | 保持不安而不是炸场 |
| 惊悚异象   | 先抽走音乐，再给异响           | 静比响更有力       |
| 情感回针   | 单音钢琴、口琴、远处广播式旋律 | 少而准，不能灌满   |

### 5.3 声音策略

1. 异响应来自环境本身，不要做廉价特效音。
2. 安静场景里只保留一个反常声音，最有效。
3. 情感段落用声音“退后”，让呼吸、纸张、衣料摩擦变清楚。

## 六、Seedance 2.0 专项适配

### 6.1 核心规则

- 所有抽象词必须转写为物理描述。
- 每镜至少写清：光源、站位、动作轨迹、关键道具状态。
- 连续镜头保持人物造型、道具破损位置、空间结构一致。

### 6.2 抽象词翻译表

| 抽象写法 | Seedance 2.0 具象写法                                            |
| -------- | ---------------------------------------------------------------- |
| 气氛压抑 | 头顶日光灯偏绿，照亮人物额头和桌面，其余墙角沉入暗部，光比约 1:5 |
| 很紧张   | 人物停在门口，右手抓住门框，指节发白，呼吸带动肩膀两次明显起伏   |
| 灯光诡异 | 走廊尽头白炽灯每 2 秒轻微闪一次，亮度衰减约 20% 后恢复           |
| 雨夜很冷 | 路面有水膜反光，人物肩头衣料被雨打深一层，冷空气让呼吸节奏变重   |

### 6.3 画质底座

1080p，真人实拍胶片感，35mm 颗粒，旧乡镇真实材质，白炽灯与荧光灯混合实用光，非 CG 非渲染

### 6.4 光效指令库

| 方案       | Seedance 光效写法                                              |
| ---------- | -------------------------------------------------------------- |
| A 灰天漫射 | 阴天天光从门窗均匀进入，色温约 5200K，低反差，墙面细节清楚可见 |
| B 白炽孤岛 | 一盏头顶白炽灯约 2800K，下打形成局部亮区，四周快速落入暗部     |
| C 荧光冷压 | 顶部日光灯约 5000K 偏绿，照亮人物额头和桌面，空间被压平        |
| D 夜市混光 | 摊位暖黄灯 3000K 为主，远处店铺冷白光 4500K 辅助，色彩脏而乱   |
| E 雨夜反射 | 路灯暖黄 3000K 打在湿地面形成反光，环境冷蓝灰 6000K 包裹       |
| F 黄昏余光 | 低角度暖光 3200K 从巷口斜入，墙面和尘土被照亮                  |
| G 守夜蓝黑 | 环境光很弱，整体偏蓝黑，仅留一处灯源勾出轮廓                   |

### 6.5 多镜头最佳实践

1. 一个片段建议 2-3 个镜头，4-10 秒内完成。
2. 开头先给空间或异常，第二镜给主人物反应，第三镜再给线索或尾钩。
3. 不要每镜都动，固定机位更适合乡镇质感。

## 七、全局叙事约束

1. **先真实，后诡异** — 空间先让观众相信，再让它出问题。
2. **先线索，后解释** — 视觉先种下东西，解释可以晚一点。
3. **情感必须附着在旧物或旧地上** — 不要凭空抒情。
4. **短剧竖屏优先中近景** — 让脸、手、道具、门缝都能被看清。
5. **惊悚不能脱离乡镇现实** — 再灵异，也要像是从这块地里长出来的。
