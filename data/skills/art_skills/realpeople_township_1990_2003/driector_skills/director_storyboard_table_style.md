# 分镜表美术手法 · 时代乡镇竖屏版

## 一、分镜表定位

本文件用于把“年代乡镇气质”翻译为可执行的分镜语句，适配竖屏短剧与视频生成模型。分镜表不是写感受，而是写镜头如何让观众感受到这些感受。

---

## 二、描述铁律——把抽象导演意图改写成 AI 可执行语言

| 禁止的抽象表达 | 可执行写法                                                                |
| -------------- | ------------------------------------------------------------------------- |
| 他很紧张       | 头向右侧缓慢转动约 30 度，视线从桌面移向门口，右手抓紧椅背，停顿约 1.2 秒 |
| 气氛很压抑     | 头顶日光灯偏绿，照亮人物额头和桌面，墙角与走廊末端沉入暗部，光比约 1:5    |
| 走廊很可怕     | 走廊尽头只剩一盏白炽灯，地面有积水反光，右上角保留整块黑暗盲区            |
| 旧物很有故事   | 账本边角翻毛，封皮压出手印油痕，中间夹着一张卷边收据                      |

## 三、竖屏运镜铁律

- **上下摇移（Tilt）**：用于空间揭示，常见于“灯泡到人物”“地面鞋印到人物反应”。
- **纵向缓推/拉**：在巷道、楼道、走廊强化压迫感，速度建议每秒 0.2-0.4 米。
- **限制横移**：竖屏快速横摇会丢信息，横移仅用于揭示盲区且速度减半。
- **手持微晃**：允许轻微不规则晃动，用于搜寻和惊惧，不宜整场都抖。
- **固定机位优先**：年代乡镇空间经得起看，固定机位更容易让细节自己发声。

---

## 四、镜头信息最小集（每镜必填）

1. 景别（特写/近景/中近景/中景/全景/空镜）
2. 运镜方式与速率
3. 主体位置（上中下 + 左中右）
4. 光源与色温（主光+辅光+光比）
5. 关键动作（轨迹+时长）
6. 关键道具状态（是否变化）
7. 可见年代锚点（灯泡/绿墙裙/旧挂历/锈门等）
8. 转场方式与环境音

## 五、光源语句法（示例）

```text
主光：头顶白炽灯，约 2800K，垂直下打，硬光。
辅光：走廊尽头天光，约 5600K，弱补。
光比：约 1:6，暗部保留轮廓。
```

```text
主光：门外路灯暖光，约 3000K，从画面右上斜入。
辅光：室内荧光灯，约 5000K，低亮度。
光比：约 1:5，形成冷暖冲突。
```

```text
主光：阴天天光，约 5200K，从门口和窗户均匀散入。
辅光：室内无额外灯源，仅墙面与地面反射补光。
光比：约 1:2，空间低反差但旧痕清楚。
```

## 六、竖屏构图规则

- **上中下三段式**：上段压迫信息（电线/灯/梁），中段主体，下段线索（鞋印/水渍/碎片/道具）。
- **盲区策略**：在画面顶部、转角或门后保留暗区，用于悬疑信息延迟。
- **前景遮挡**：铁门边缘、玻璃反光、晾衣杆、门框，都可增强窥视感。
- **人物不必居中**：人物放偏，空出来的一侧更适合悬疑和等待。

## 七、节奏模板（短剧）

| 段落     | 镜头长度建议 | 任务               |
| -------- | ------------ | ------------------ |
| 开场钩子 | 1.0s-2.5s    | 异常道具/异响/人影 |
| 搜证推进 | 1.5s-3.0s    | 信息累积           |
| 悬疑停顿 | 3.0s-5.0s    | 等待与压迫         |
| 反应镜头 | 1.0s-2.0s    | 人物微表情         |
| 情感回针 | 2.0s-4.0s    | 旧物 + 表情        |
| 收束过渡 | 2.0s-4.0s    | 空镜或道具特写     |

## 八、环境动态——让画面不是死的

| 场景        | 可描述动态                                 |
| ----------- | ------------------------------------------ |
| 走廊/室内   | 灯泡轻闪、门帘轻动、风吹报纸边角、雨声打窗 |
| 夜市/街口   | 塑料棚被风压动、锅气上升、摩托慢速掠过     |
| 雨夜巷道    | 水滴从屋檐滴下、路面积水被风扰起涟漪       |
| 宿舍/值班室 | 热水壶微冒气、台灯边缘有飞虫、纸张被手带起 |

### 动态规则

1. 动态必须有来源，例如风、雨、人、灯管老化。
2. 动态幅度要小，乡镇空间更适合“慢慢不对劲”，不是大场面炫技。
3. 动态与情绪同步：紧张时加快或突停，情感段落则让动作更慢。

## 九、动作描述标准

禁止写“他很紧张地看向门口”。

改写为：

```text
头向右侧缓慢转动约 30 度，视线从桌面移向门口，
右手抓紧椅背，停顿约 1.2 秒。
```

禁止写“她翻了翻账本”。

改写为：

```text
左手按住账本封皮，右手从右下角掀起两页，
纸张翻动约 0.8 秒后停住，目光落在中缝夹着的收据上。
```

禁止写“他打开了门”。

改写为：

```text
右手压下门把手约 30 度，门向内缓慢推开约 50 厘米，
人物身体停在门外半步，先探头再进身。
```

## 十、空间坐标与衔接规则

### 10.1 站位标注

- 水平位置：画面左侧三分之一 / 中央 / 右侧三分之一
- 纵深位置：距镜头约多少米，属于前景/中景/背景
- 固定物参考：距铁门约 0.5 米、距窗台约 1 米、距桌边约 30 厘米

### 10.2 衔接原则

1. 前后镜头必须说清人物是否移动。
2. 若人物移动，道具也要跟着变位置。
3. 同一空间切换机位时，主要门窗和道具方向不能突然翻转。

## 十一、运镜与转场规范

### 11.1 允许的运镜

| 运镜     | 速率建议        | 适用场景                 |
| -------- | --------------- | ------------------------ |
| 固定     | 无              | 等待、对话、空镜         |
| 手持微晃 | 幅度约 ±2cm     | 搜寻、惊惧、贴身跟随     |
| 缓推     | 每秒 0.2-0.4 米 | 逼近线索、情绪收紧       |
| 缓拉     | 每秒 0.2-0.4 米 | 揭示环境、收束段落       |
| Tilt     | 缓慢            | 灯/梁到人，地面到线索    |
| 慢横移   | 低速            | 柜门后、门框后、帘后揭示 |

### 11.2 允许的转场

- 硬切：同一场戏默认方案
- 空镜过渡：留情绪缓冲和空间说明
- 道具匹配转场：旧照片、账本、灯泡等可作为连接器
- 光影匹配转场：黄昏墙面光斑切到另一处相似光源

### 11.3 禁止的转场

- 花哨特效转场
- 无叙事意义的快速甩镜
- 连续使用多个炫技转场破坏年代真实感

## 十二、音画同步规划

| 场景          | 建议环境音                         |
| ------------- | ---------------------------------- |
| 走廊          | 灯管嗡鸣、脚步回声、门轴声         |
| 夜市          | 人声、锅铲声、塑料棚摩擦、远处摩托 |
| 宿舍          | 热水壶、布料摩擦、床架轻响         |
| 雨夜巷道      | 雨滴、鞋踩水、远处狗叫             |
| 派出所/卫生院 | 翻纸声、日光灯、电流声、走廊回音   |

关键节点建议标明时间点：

- `t=1.2s` 旧门轴响一下
- `t=2.8s` 账本中夹着的收据露出边角
- `t=4.0s` 走廊尽头灯泡轻闪一次

## 十四、空镜使用规范

空镜不是“没人”，而是“空间里还留着事”。

| 空镜类型     | 线索                   | 用途                   |
| ------------ | ---------------------- | ---------------------- |
| 刚离场空镜   | 灯没关、椅子歪、烟未灭 | 让观众知道刚发生过什么 |
| 长期封存空镜 | 灰层、卷边纸张、潮斑   | 旧案、旧屋、被遗弃感   |
| 雨夜空镜     | 水膜、灯反光、空巷     | 情绪加压、切入危险     |
| 黄昏空镜     | 斜阳打在旧墙、风吹门帘 | 情感回针、回忆引子     |

---

## 十五、自检清单

| 检查项       | 通过标准                           |
| ------------ | ---------------------------------- |
| 光源可追溯   | 每镜都能说清光从哪来               |
| 时代锚点足够 | 至少有 1-2 个明确乡镇年代元素      |
| 动作可执行   | 每个动作有轨迹和时长               |
| 连续性成立   | 服装、道具、空间衔接无跳变         |
| 盲区存在     | 悬疑镜头至少保留一处未完全揭示区域 |
| 空镜有信息   | 不是纯空白，而是留有生活或事件痕迹 |
