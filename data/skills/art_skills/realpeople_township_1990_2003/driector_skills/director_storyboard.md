# 分镜提示词 · 真人时代乡镇 · 风格专属技法

---

## 适用范围

本 Skill 专用于 **80 年代后期至 2003 年前中国乡镇/城乡结合部** 的真人分镜提示词创作，适配年代悬疑、乡镇现实主义、灵异民俗、成长创伤、旧案追查等题材，默认服务竖屏短剧与短视频分镜，但其中的镜头语法同样可用于横屏项目。

---

## 一、人物情绪到镜头表达

> 时代乡镇的情绪不是靠夸张表演，而是靠脸上“收着”的东西。眼神、嘴角、呼吸、肩颈、手部小动作，远比大哭大喊更有劲。

| 情绪输入 | 面容描述                   | 眼神描述                 | 微表情补充           | 分镜提示核心             |
| -------- | -------------------------- | ------------------------ | -------------------- | ------------------------ |
| 疲惫戒备 | 眼皮微沉，下颌收紧         | 盯着某处不完全放心       | 肩部轻绷，嘴唇发干   | 近景，手持微晃，冷光侧打 |
| 惊惧     | 面部僵住，表情停半秒       | 瞳孔收紧或骤然聚焦       | 呼吸打断，嘴唇轻启   | 极近景，短时静止后突变   |
| 迟疑     | 表情不满格，嘴角想说又收回 | 视线游移于门口和物件之间 | 手部有犹豫动作       | 中近景，缓推，留盲区     |
| 心虚     | 眉眼发紧，不敢完全抬头     | 看一眼就躲               | 下巴往回收           | 中景，前景遮挡，暗部多   |
| 决绝     | 目光定住，不再回避         | 向前钉住                 | 动作突然变简单、变硬 | 中景转近景，轮廓硬光     |
| 麻木     | 面部松掉，情绪像被抽空     | 看着前方但没真正聚焦     | 肩垮、呼吸浅         | 固定近景，弱光，低反差   |
| 暂时温柔 | 表情放软但不甜             | 目光停在旧物或旧人上     | 手指动作更轻         | 近景，暖光小范围包裹     |
| 压抑愤怒 | 面部发硬但不爆发           | 盯人不眨或极慢眨眼       | 咬肌用力，手握紧     | 中近景，荧光冷压或硬侧光 |

## 二、摄影机前的光——乡镇空间里的真实光源

### 2.1 时段光线

| 时段     | 光源行为                       | 色温倾向               | 适合情绪               |
| -------- | ------------------------------ | ---------------------- | ---------------------- |
| 清晨     | 冷中带暖的散天光，空气潮       | 4200K-5200K            | 后怕、未醒、真实日常   |
| 白天阴天 | 散射、低反差、墙体纹理清楚     | 5000K-5600K            | 调查、观察、压着的生活 |
| 正午     | 顶光硬，阴影短，热与压迫明显   | 5600K-6500K            | 冲突前、暴躁、暴露感   |
| 黄昏     | 低角度暖光，长阴影，尘土被照亮 | 3000K-3800K            | 回忆、诀别、情感回针   |
| 深夜     | 白炽灯、路灯、手电筒构成孤岛   | 2600K-3200K / 局部冷光 | 惊悚、守夜、秘密交换   |

### 2.2 情绪光影

| 情绪基调 | 光线配方                | 摄影约束                       |
| -------- | ----------------------- | ------------------------------ |
| 真实日常 | 灰天漫射或窗边散光      | 保留材质纹理，低反差           |
| 制度压迫 | 偏绿荧光灯顶打          | 空间压扁，人物显得更累         |
| 烟火误导 | 暖脏混光，冷光局部穿刺  | 允许色彩不“高级”，但要真实     |
| 雨夜悬疑 | 暖黄路灯 + 冷湿环境反射 | 地面反光、空气潮气必须出现     |
| 情感停顿 | 黄昏余光或单一室内暖灯  | 光落在脸和旧物上，其余不必全亮 |

## 四、摄影机语法——镜头与运镜选择

### 4.1 焦段选择

| 焦段    | 叙事性格                     | 适用场景                 |
| ------- | ---------------------------- | ------------------------ |
| 24-28mm | 空间压人，纵深强，逼仄感明显 | 走廊、深巷、宿舍、夜路   |
| 35mm    | 人与环境共存，纪实感强       | 跟随、调查、对话         |
| 50mm    | 最像人眼，看脸和道具都稳     | 关键对话、人物观察       |
| 85mm    | 压缩空间，突出表情和线索     | 惊惧反应、物件特写、窥视 |

### 4.2 景别选择

| 景别   | 取景范围                     | 用途                 |
| ------ | ---------------------------- | -------------------- |
| 极近景 | 眼睛、嘴角、手指、按键、纸角 | 线索显影、情绪瞬断   |
| 近景   | 头顶至胸部                   | 微表情、心理波动     |
| 中近景 | 头顶至腰部                   | 对话、动作、手部行为 |
| 中景   | 半身到膝上                   | 人与空间关系、对峙   |
| 全景   | 人物全身 + 环境              | 交代空间、突出渺小   |
| 空镜   | 无人或人极小                 | 情绪缓冲、环境压迫   |

### 4.3 运镜选择

| 运镜      | 视觉心理                 | 适用场景                 |
| --------- | ------------------------ | ------------------------ |
| 固定      | 冷静、让观众自己找不对劲 | 等待、对话、空镜         |
| 手持微晃  | 在场感、神经紧绷         | 搜索、惊惧、跟随         |
| 纵向缓推  | 逼近线索或人物内心       | 发现、判断、下决心       |
| 缓拉      | 抽离、揭示更大空间       | 真相落地、情绪后撤       |
| Tilt 摇移 | 竖屏最有效的空间揭示     | 从灯泡到人、从鞋印到表情 |
| 横移揭示  | 慢速揭露盲区             | 门后、帘后、柜子后       |

## 五、固定风格锚定词

### 摄影锚定

真人实拍摄影，时代乡镇电影质感，80年代后期到2003年前中国乡镇，胶片颗粒，白炽灯与荧光灯实用光，非CG非渲染

### 人物质感锚定

真实皮肤纹理、可见毛孔和轻微粗糙、发丝碎乱、旧衣料有起球和褶皱、自然微表情、真实肢体语言

### 场景质感锚定

绿墙裙、返潮墙面、旧铁门、水磨石地面、灰尘和生活痕迹、真实乡镇空间、非现代化改造

### 画质锁定词

模式A（中文）：
真人摄影高清画质，电影级画面质感，画面无字幕、无水印、无标题叠字、无UI元素

模式B（英文）：
live-action township cinematic photography, real skin texture, visible wear and tear, 35mm film grain, no subtitles, no watermark, no title overlay, no UI

### 负向词模板（英文）

no 3D render, no CGI, no anime, no cyberpunk, no modern fashion, no smart devices,
no showroom clean interior, no plastic skin, no luxury styling, no futuristic city, no watermark

## 六、美学禁止项

- 禁止出现现代高楼 CBD、现代商场、智能手机时代视觉符号。
- 禁止韩式精修、时尚大片光位、过度高级感色彩。
- 禁止古风、仙侠、民国、赛博朋克、科幻、欧洲小镇等越界世界观。
- 禁止 CG 术语和动画媒介感。
