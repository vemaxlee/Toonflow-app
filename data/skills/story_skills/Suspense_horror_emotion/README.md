# 悬疑推理主导 · 惊悚灵异辅助 · 情感点缀 · 导演叙事手法技能包

## 简介

本技能包服务于 **悬疑推理为主（60%）、恐怖灵异为辅（30%）、情感共鸣点缀（10%）** 的复合类型故事创作。它不是把三个类型简单拼接在一起，而是要求导演在同一条叙事链上同时完成三件事：

- 用推理结构持续驱动观众追问“真相到底是什么”
- 用灵异或惊悚表象不断干扰判断，制造生理上的不安与心理上的失衡
- 用克制的情感回针解释人物为何非查不可、为何无法放下、为何真相会刺痛人心

本技能包适用于长剧、短剧、章节式互动影像、悬疑短片、剧情向 AI 视频与分镜规划场景。视觉风格可自由替换，但叙事比例与镜头心理学必须保持稳定。

## 核心理念

- **推理骨架必须成立**：所有灵异异象、怪谈传闻、诡异画面，都必须最终服务于线索推进、嫌疑塑造、误导策略或主题揭示，不能沦为脱离主线的单次惊吓。
- **恐怖不是替代逻辑，而是扰乱逻辑**：恐怖段落的职责是拉高不确定性，让观众和主角一度怀疑“现实规则是否已经失效”，但不能破坏整体推演的可回溯性。
- **情感只做重击，不做铺满**：情感只占 10%，因此必须集中在最疼的地方出现，例如遗物、旧约、亏欠、保护欲、无法说出口的告别。少，但要准。
- **镜头本身可以说谎**：通过景别遮蔽、视线误导、声音先行、转场错位，让观众在相信镜头的同时被镜头欺骗，最终在回看时发现每个细节都曾出现。

## 类型配比原则

| 类型模块     | 占比目标 | 导演任务               | 常用手段                                   | 失衡风险                           |
| ------------ | -------- | ---------------------- | ------------------------------------------ | ---------------------------------- |
| **悬疑推理** | 60%      | 保持案件/谜团持续推进  | 信息差、红鲱鱼、线索回收、视角控制         | 若不足，故事只剩氛围，没有追看动力 |
| **恐怖灵异** | 30%      | 制造未知压迫、扰乱判断 | 异响、死角、反常空间、非同步影像、民俗意象 | 若过量，会冲垮逻辑，变成纯惊吓片   |
| **情感共鸣** | 10%      | 提供人物动机与结尾回响 | 遗物、闪回、迟到的解释、守护关系、创伤记忆 | 若过量，会稀释悬疑节奏，转向苦情戏 |

### 配比执行规则

1. **每个关键段落都以“问题”起步**：哪怕先出现灵异画面，观众最终也要回到“这意味着什么”“谁做的”“为什么现在发生”。
2. **每轮惊悚都必须换来推理收益**：一次鬼影、一次怪声、一次诡异目击，至少要带出一个新线索、一个新误导，或一个旧信息的重解释。
3. **情感不占篇幅，占节点**：只在动机揭示、关系断裂、真相落地、余韵收束等节点使用，避免中段频繁煽情打断追查。

## 使用方式

1. **先定主谜团与情感根部**：先回答“主角到底在查什么”“查下去会碰到什么”“为什么这件事伤到他/她”，再设计灵异壳层。
2. **用叙事规划文件搭结构**：确定每个章节或每集的疑点、误导、惊悚点、情感回针与真相释放顺序。
3. **用分镜表文件落镜头**：把上一步的结构转换为可执行的景别、运镜、时长、声音与转场决策。
4. **复查类型配比**：检查是否出现“鬼吓人过多但谜团不推进”或“情绪铺陈过长导致悬疑失速”的情况。

## 适用创作场景

- 乡镇/县城/旧宅/民俗空间中的悬疑灵异故事
- 偏现实主义外壳、但允许超自然误导的案件调查叙事
- 单元怪谈式长线主谜团结构
- 章节化短剧、互动剧、分段式 AI 视频项目

## 使用禁忌

1. 不要把恐怖桥段当作独立节目插入，必须与线索或人物关系绑定。
2. 不要用大段解释台词代替镜头揭示，尤其在反转时。
3. 不要让情感戏频繁打断调查，情感应当像针一样刺入，而不是像水一样铺开。
4. 不要让“真有鬼”成为偷懒答案，除非项目世界观从一开始就允许且能自圆其说。
